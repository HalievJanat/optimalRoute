Автоматизированная система расчета оптимального маршрута движения автотранспорта по заданному критерию

Введение

Автоматизированная система расчета оптимального маршрута движения автотранспорта по заданному критерию будет использоваться для определения наилучшего маршрута для транспортных средств с учетом различных факторов.

Программа использует базу данных MS SQL Server (требует предустановление сервера MS SQL). Программная система совместима с Windows 7 и выше. Системные требования: 3Gb оперативной памяти, 23Gb жесткого диска.

Авторизация, запуск программы

Данная система состоит из двух частей: режим пользователя и режим админа. При запуске системы открывается главная страница. После загрузки стартовой страницы необходимо войти или зарегистрироваться в систему, переход осуществляется соответствующими элементами меню.

На странице войти, необходимо ввести логин и пароль для входа в систему. При нажатие на кнопку «Войти» программа проверяет введенные учетные записи и определить роль (администратор, пользователь).

Регистрация

При нажатие на «Зарегистрироваться» осуществляется переход на страницу регистрации. Необходимо ввести уникальный логин и пароль для сохранения учетных данных в систему. Проверка уникальности и сохранения данных в систему будет осуществляться после нажатие кнопки «Зарегистрироваться».

Функции администратора:

  • Авторизация пользователя в системе (ввод логина и пароля)
  • Просмотр справочной информации о системе
  • Создание структуры улично-дорожной сети:
    • Добавление/удаление перекрестка
    • Добавление/удаление прогона
    • Перемещение вершин
    • Настройки параметров перекрестка:
      • Установка светофора
      • Настройка длительности светофорных фаз
    • Настройки параметров прогонов:
      • Выбор названия улицы
      • Выбор направления движения
      • Установка длины участка
      • Расстановка знаков дорожного движения, устанавливающие ограничения скорости
      • Выбор типа покрытия
      • Установка полицеского
  • Работа с базой данных:
    • Добавление сущностей
    • Редактирования сущностей
    • Удаление сущностей
  • Ведение справочников:
    • Улиц
    • Типов покрытия
    • Степени коррумпированности полицеского
    • Типов штрафов
    • Видов топлива
  • Работа с водителями:
    • Ввод ФИО
    • Выбор типа водителя
    • Удаление транспортного средства
  • Работа с транспортным средством:
    • Ввод марки
    • Выбор вида топлива
    • Ввод расхода топлива (л/100км)
    • Ввод максимально развиваемой скорости

Функции пользователя:

  • Регистрация пользователя в системе (ввод логина, пароля)
  • Просмотр справочной информации о системе
  • Авторизация пользователя в системе (ввод логина и пароля)
  • Выбор структуры улично-дорожной сети
  • Выбор критерия поиска
  • Выбор точки отправления
  • Выбор точки прибытия
  • Выбор водителя
  • Запуск моделирования

Админ

На данной странице осуществляется добавление на экран элементы УДС (перекресток, прогон).

Перекресток устанавливается путем выбора его на левой панели и кликом левой кнопки мыши на сетку.

Прогон устанавливается путем выбора его на левой панели и выбора двух перекрестков между которыми будет проведен прогон.

Удаление и добавление параметров происходит через выбор нужного пункта в контекстном меню, которое появляется при нажатии правой кнопки мыши на элемент УДС. Просмотр параметров элемента УДС возможен после нажатия двойным кликом левой кнопки мыши на этот элемент.

Перемещение перекрестка осуществляется выбором соответствующего пункта контекстного меню и после нажатием на сетку левой кнопкой мыши.

Создать карту

При нажатии на пункт меню «Создать новую карту» будет происходить очистка сетки и переход к начальному состоянию или будет появляться модальное окно с предупреждением о том, что карта не сохранена.

Открыть карту

При нажатии на пункт меню «Открыть карту» будет появляться модальное окно, в котором можно выбрать нужную УДС для загрузки или будет появляться модальное окно с предупреждением о том, что карта не сохранена.

Сохранить карту

При выборе пункта меню «Сохранить» будет появляться модальное окно, в котором нужно будет ввести уникальное название УДС для её дальнейшего сохранения в базе данных.

БД

При выборе пункта меню «БД» будет происходить переход на новую страницу для добавления информации в базу данных.

При выборе одного из пунктов на левой панели. После будут загружаться необходимые поля для внесения данных. Для редактирования или добавления данных необходимо выбрать на правой панели соответствующий пункт. В режиме добавления при нажатии на кнопку «Добавить» система проверяет данные и сохраняет их в БД. В режиме изменения осуществляется удаление и редактирования данных. Для этого необходимо нажать кнопку (стрелочку) справа от таблицы и выбрать соответствующий пункт. Изменение данных осуществляется в таблице. Для фиксации изменения необходимо нажать на кнопку "Сохранить". При удаление, если элемент связан с другими элемента выводится модальное окно с подтверждением удалением всех элементов. При выборе пункта меню «Вернуться к построению карты» происходит загрузка главной страницы пользователя в режиме администратора.

О системе

При выборе пункта меню «О системе» будет открываться новая страница с информацией об использовании системы.

О разработчиках

При выборе пункта меню «О разработчиках» будет появляться всплывающее окно с информацией о разработчиках системы.

Пользователь

Для создания оптимального маршрута необходимо выбрать точку отправления, точку прибытия, водителя, а также критерий определения оптимального маршрута. Для этого выбирается соответствующий пункт на панели слева.

При нажатие на пункт «Выбор точки отправления и прибытия», правой кнопкой мыши на карте выбираются два перекрестка, первый является пунктом отправления, а второй пунктом прибытия. При нажатии больше двух раз выбор чередуется.

При нажатие на пункт «Выбор автомобиля» появляется поле с выпадающим списком о водителях, при выборе которого полная информация о данном водителе будет отражена на правой панели страницы.

В разделе «Выбор критерия» будут представлены различные критерии для построения маршрута, в котором осуществляется выбор одного из них.

При нажатие на кнопку «Построить» осуществляется визуализацию пути на карте.

Загрузить карту

При нажатии на пункт меню «Загрузить карту» появляется модельное окно, в котором можно выбрать нужную карту для загрузки. Открытие выбранной карты осуществляется выбором соответствующего пункта в окне.

Справочная информация

При выборе пункта меню «Справочная информация» откроется новая страница с информацией об использовании системы.
