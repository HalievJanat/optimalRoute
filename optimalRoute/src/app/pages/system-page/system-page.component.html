Автоматизированная система расчета оптимального маршрута движения автотранспорта по заданному критерию

Введение

Автоматизированная система расчета оптимального маршрута движения автотранспорта по заданному критерию будет использоваться для определения наилучшего маршрута для транспортных средств с учетом различных факторов

Программа использует базу данных MS SQL Server (требует предустановление сервека MS SQL). Программная система совместима с Windows 7 и выше. Системные требования: 500Mb оперативной память, 500Mb жесткого диска.

Авторизация, запуск программы

Данная система состоит из двух частей: режим пользователя и режим админа. При запуске системы открывается главная страница. После загрузки стартовой страницы необходимо войти или зарегистрироваться в систему, переход осуществляется соответствующими элементами меню.

На странице войти, необходимо ввести логин и пароль для входа в систему. При нажатие на кнопку «Войти» программа проверяет введенные учетные записи и определить роль (администратор, пользователь).

Регистрация

При нажатие на «Зарегистрироваться» осуществляется переход на страницу регистрации. Необходимо ввести уникальный логин и пароль для сохранения учетных данных в систему. Проверка уникальности и сохранения данных в систему будет осуществляться после нажатие кнопки «Зарегистрироваться».

Админ

На данной странице осуществляется добавление на экран элементы УДС (перекресток, прогон).

Перекресток уставливается путем выбора его на левой панели и кликом левой кнопки мыши на сетку.

Прогон уставливается путем выбора его на левой панели и выбора двух перекрестков между которыми будет проведен прогон.

Удаление и добавление параметров происходит через выбор нужного пункта в контекстном меню, которое появляется при нажатии правой кнопки мыши на элемент УДС. Просмотр параметров элемента УДС возможен после нажатия двойным кликом левой кнопки мыши на этот элемент.

Создать карту

При нажатии на пункт меню «Создать новую карту» будет происходить очистка сетки и переход к начальному состоянию или будет появляться модальное окно с предупреждением о том, что карта не сохранена.

Открыть карту

При нажатии на пункт меню «Открыть карту» будет появляться модальное окно, в котором можно выбрать нужную УДС для загрузки или будет появляться модальное окно с предупреждением о том, что карта не сохранена.

Сохранить карту

При выборе пункта меню «Сохранить» будет появляться модальное окно, в котором нужно будет ввести уникальное название УДС для её дальнейшего сохранения в базе данных.

БД

При выборе пункта меню «БД» будет происходить переход на новую страницу для добавления информации в базу данных.

При выборе одного из пунктов на левой панели страницы будут загружаться необходимые поля для внесения данных. Для изменения, добавления или удаления данных необходимо выбрать на правой панели соответсвующий пункт. В режиме удаление при нажатии на кнопку «Добавить» система проверяет данные и сохраняет их в БД. В режиме изменения осуществляется удаление и редактирования данных. Для этого необходимо нажать кнопку (стрелочку) спарава от таблицы и выбрать соответсвующий пункт. Изменение данных осуществляется в таблице и для фиксации изменения необходимо нажать на кнопку "Сохранить". При удаление, если элемент связан с другими элемента выводится модальное окно с подтверждением удалением всех элементов. При выборе пункта меню «Вернуться к построению карты» происходит загрузка главной страницы пользователя в режиме администратора.

О системе

При выборе пункта меню «О системе» будет открываться новая страница с информацией об использовании системы.

О разработчиках

При выборе пункта меню «О разработчиках» будет появляться всплывающее окно с информацией о разработчиках системы.

Пользователь

Для создания оптимального маршрута необходимо выбрать точки отправления и прибытия, водителя, а также критерий определения оптимального маршрута. Для этого выбирается соответствующий пункт на панели слева.

При нажатие на пункт «Выбор точки отправления и прибытия», правой кнопкой мыши на карте выбираются два перекрестка, первый является пунктом отправления и второй пунктом прибытия. При нажатие больше двух раз выбор чередуется.

При нажатие на пункт «Выбор автомобиля» появлется поле с выпадающим списком с информацией о водителях, при выборе которого полная информация о данном водителя будет отражена на правой панели страницы.

В разделе «Выбор критерия» будут представлены различные критерии для построения маршрута, в которой осуществляется выбор одного из них.

При нажатие на кнопку «Построить» осуществляется визуализацию пути на карте.

Загрузить карту

При нажатии на пункт меню «Загрузить карту» будет появляться модельное окно, в котором можно выбрать нужную карту для загрузки. Открытие выбранной карты осуществляется выбор соответствующего пункта в окне.

Справочная информация

При выборе пункта меню «Справочная информация» будет открываться новая страница с информацией об использовании системы.
